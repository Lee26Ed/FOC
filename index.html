Fundamentals of Computing

This course is an introduction to the fundamentals of Computer Science that will provide students with a foundation for future courses in Computer Science. The course is a broad survey of the discipline through topics that allow students to understand the capabilities of computer systems in terms of: the computer architecture; operating systems and computer networks; software and software development; data organization and visualization; and the potential of computing machines.

Projects

Below are 9 different simple interactive projects created using HTML, CSS and JavaScript. Feel free to try them out and see what you think.

Project 1: Not Gate

NOT gate is a digital logic gate with 1 input and 1 output that operates an inverter operation of the input.The output of the NOT gate is the reverse of the input. When the input of the NOT gate is true then the output will be false and vice versa.

Project 2: And Gate

AND gate is a digital logic gate with 2 inputs 1 output, which perform logical conjunction based on the combinations of its inputs.The output of this gate is true only when all the inputs are true. When one or more inputs of the AND gate is false, then the output of the AND gate is false.

Project 3: Or Gate

OR gate is a digital logic gate with 2 inputs and 1 output, that performs a logical conjunction based on the combinations of its inputs.The output of the OR gate is true when one or more inputs are true. If all the inputs of the OR gate are false, then the output of the OR gate is false.

Project 4: XOR Gate

XOR gate (pronounced as Exclusive OR) is a digital logic gate with 2 inputs and 1 output that gives a true output when one, and only one, of the inputs to the gate is true. If both inputs are false or both are true, a false output results.

Project 5: Binary to Hexadecimal Converter

The Binary to Hexadecimal Converter takes any 4 digit binary number and converts it to its Hexadecimal form.

Project 6: ASCII Converter

The ASCII Converter converts any number between 0 and 127 to its character form.

Project 7: UNICODE Converter

The UNICODE Converter takes in a UNICODE number and converts it into its character form.

Project 8: Machine Language Instructions Converter

The Machine Language Instructions Converter is a simple model of a machine language.

Project 9: BMI Converter

Body Mass Index (BMI) Calculator is a measure of body fat based on height and weight that applies to adult men and women.
